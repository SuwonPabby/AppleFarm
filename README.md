# 🍎 AppleFarm - Gesture Recognition🖐


## 🙂 What we made?
- webcam 을 이용한 실시간 gesture recognition model

```  
    1. Palm
    2. l
    3. fist
    4. fist_moved
    5. thumb
    6. index
    7. ok
    8. palm_moved
    9. c
    10. down
```
